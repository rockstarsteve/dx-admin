# 基础开发项目

#### 介绍
```
单体项目，做到了解耦，多套部署，二次开发cloud的考虑。
解决别人框架封装太深，在快速开发中无法进行快速定制化修改的问题
```



#### 软件架构
- 开发环境（必要的）
    - 后台
      - jdk1.8
    - 前端
      - node8+
        
- 前端：
    - vue-element-template（vue-cli 4.x）
    - element-ui
    
- 后端：
    - springboot2.x  
    - mybatis plus3.x
    - jwt
    - 
- 数据库：mysql5.7
